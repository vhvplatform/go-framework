# Server - Golang Backend

This directory contains the Golang microservices for the backend of the platform.

## Structure

```
server/
├── docker/           # Docker Compose configurations
├── scripts/          # Automation scripts
├── mocks/            # Mock services for development
├── tools/            # Development tools and CLI
├── k8s/              # Kubernetes configurations
├── configs/          # Configuration files
├── fixtures/         # Test fixtures and sample data
├── postman/          # API testing collections
└── examples/         # Example implementations
```

## Getting Started

Refer to the main documentation in the `/docs` directory for setup and development instructions.

## Technology Stack

- Go 1.25+
- Docker & Docker Compose
- Kubernetes
- MongoDB, Redis, RabbitMQ
